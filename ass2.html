Paired Tags

  2. <b>Bold Text</b>
  3. <i>Italic Text</i>
  4. <u>Underlined Text</u>
  5. <h1>Heading </h1>

Unpaired Tags

  1. <br> (Line Break)
  2. <hr> (Horizontal Line )
  3. <img src="image.jpg" alt="Example">
  4. <input type="text">
